Secure HTTP (HTTPS)

Last updated {{ scan_date | date("%B %d, %Y") }}

This data measures whether Jamaican government domains support the HTTPS protocol (https://), and the strength of that support. HTTPS provides a secure connection across the internet between websites and their visitors, and is becoming the new baseline for public web services.

Note that HTTPS generally does not affect whether a website is vulnerable to hacking. For more information on what HTTPS does (and doesn't do), visit the HTTPS FAQ.
